    Discrete EVent system Specification for Ruby

    This is a meta-distribution of DEVS including all currently available plugins

    • http://github.com/romain1189/devs_ruby

    Examples

    require 'devs_ruby'

    Dependencies

    • DEVS (>= 0.5)
    • DEVS::Models (>= 0.5)
    • DEVS::Examples (>= 0.5)
    • DEVS::Ext (>= 0.5)

    Installation

    The recommended installation method is via RubyGems. To install the latest official release of the gem, do:

    % [sudo] gem install devs_ruby
